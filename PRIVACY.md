# Privacy Policy

## the short version

this extension doesn't collect anything. nothing. zero.

## the longer version (for mozilla)

**data collection:** none

**data storage:** your theme preference is stored locally using firefox's built-in storage API. this never leaves your browser.

**network requests:** this extension makes zero network requests. it doesn't phone home, doesn't have analytics, doesn't track anything.

**permissions explained:**
- `storage` - to remember which theme you picked
- `activeTab` - to apply the theme CSS to digg.com pages

that's it. no funny business.
